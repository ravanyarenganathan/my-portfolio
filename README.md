# Portfolio Website

A simple portfolio website created using HTML and CSS.

## Files

- `index.html` – Main page of the portfolio website.
- `contact-form.html` – Contact page with a form for users to send messages.
- `style.css` – Contains all the styling for the website.

## Features

- Personal portfolio homepage
- Contact form
- Simple and clean design
- Built with basic HTML and CSS

## Technologies Used

- HTML5
- CSS3

## How to Run

1. Download or clone the project.
2. Open `index.html` in your web browser.
